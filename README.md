# 📖 Quranic Chatbot

AI-powered chatbot for Islamic Q&A using Quranic verses with RAG.

## Features
- 🤖 AI answers with Surah/Ayah references
- 💬 Chat interface  
- ⚡ Fast vector search
- 📱 Responsive

## Tech Stack
- Streamlit (Frontend)
- LangChain (RAG)
- Groq (LLM)
- FAISS (Vector DB)

## Quick Start
```bash
git clone https://github.com/UmerAfzalKhan/Quran-chatbot.git
cd Quran-chatbot
pip install -r requirements.txt
streamlit run src/app.py
